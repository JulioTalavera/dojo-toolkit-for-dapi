dojox.form.FileInputBlind (version 1.7)

Object » FileInputAuto » dojox.form.FileInputBlind

Usage:

var foo = new dojox.form.FileInputBlind();

An extended version of dojox.form.FileInputAuto that does not display an input node, but rather only a button and otherwise behaves just like FileInputAuto

Method Summary

  • _fixPosition() in this case, set the button under where the visible button is
  • constructor() returns dojox.form.FileInputBlind A reference to the constructor function for this object.
  • reset(e) onclick, we need to reposition our newly created input type="file"
  • startup() hide our fileInput input field
