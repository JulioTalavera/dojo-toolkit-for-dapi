dojox.form.manager._ClassMixin
Defined by dojox.form.manager

This mixin provides unified way to check/add/remove a class of controlled elements. It should be used together with dojox.form.manager.Mixin.

dojox.form.manager._DisplayMixin
Defined by dojox.form.manager

This mixin provides unified show/hide functionality for controlled elements (indicated by dojoAttachPoint attribute). Essentially it provides a way to change "style.display" parameter of controlled nodes. It should be used together with dojox.form.manager.Mixin.

dojox.form.manager._EnableMixin
Defined by dojox.form.manager

This mixin provides unified enable/disable functionality for form widgets and form elements. It should be used together with dojox.form.manager.Mixin.

dojox.form.manager._FormMixin
Defined by dojox.form.manager

This mixin adds automated "onreset", and "onsubmit" event processing if we are based on a form node, defines onReset(), onSubmit(), reset(), submit(), and isValid() methods like dijit.form.Form. It should be used together with dojox.form.manager.Mixin.

dojox.form.manager._Mixin
Defined by dojox.form.manager

This mixin provideas a foundation for an enhanced form functionality: unified access to individual form elements, unified "onchange" event processing, general event processing, I/O orchestration, and common form-related functionality. See additional mixins in dojox.form.manager namespace.

dojox.form.manager._NodeMixin
Defined by dojox.form.manager

This mixin provideas a foundation for an enhanced form functionality: unified access to individual form elements, unified "onchange" event processing, and general event processing. It complements dojox.form.manager._Mixin extending the functionality to DOM nodes.

dojox.form.manager._ValueMixin
Defined by dojox.form.manager

This mixin adds unified access to form widgets and form elements in terms of name-value regardless of the underlying type of an element. It should be used together with dojox.form.manager.Mixin.
