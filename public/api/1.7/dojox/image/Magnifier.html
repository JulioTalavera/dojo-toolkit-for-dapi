dojox.image.Magnifier (version 1.7)

Usage:

var foo = new dojox.image.Magnifier();

An unobtrusive way to add an unstyled overlay above the srcNode image element. The overlay/glass is a scaled version of the src image (so larger images sized down are clearer).

over-ride the _createGlass method to create your custom surface, being sure to create an img node on that surface.

Method Summary

  • _createGlass() create the glassNode, and an img on a dojox.gfx surface
  • _placeGlass(e) position the overlay centered under the cursor
  • _setImage(e) set the image's offset in the clipping window relative to the mouse position
  • constructor() returns dojox.image.Magnifier A reference to the constructor function for this object.
