Usage:

var foo = new dojox.atom.widget.PeopleEditor();

An editor for dojox.atom.io.model.Person objects. Displays multiple rows for the respective arrays of people. Can add/remove rows on the fly.

Method Summary

  • _add() Adds a new block of blank editors to represent a Person.

  • _createEditors(name, email, uri, index, widgetName) creates editor boxes (textbox widgets) for the individual values of a Person.
