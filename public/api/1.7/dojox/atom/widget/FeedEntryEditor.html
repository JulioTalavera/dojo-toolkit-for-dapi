dojox.atom.widget.FeedEntryEditor (version 1.7)

Object » FeedEntryViewer » dojox.atom.widget.FeedEntryEditor
dojo.require("dojox.atom.widget.FeedEntryEditor");
Defined in dojox/atom/widget/FeedEntryEditor.js

Usage:

var foo = new dojox.atom.widget.FeedEntryEditor();

An ATOM feed entry editor that allows viewing of the individual attributes of an entry.

Flag denoting if the current entry is editable or not.

Saves edits submitted when the 'save' button is pressed. Distinguishes between new and existing entries and saves appropriately.
