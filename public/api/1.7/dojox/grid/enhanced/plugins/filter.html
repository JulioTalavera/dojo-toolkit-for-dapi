dojox.grid.enhanced.plugins.filter._DataExpr
Defined by dojox.grid.enhanced.plugins.filter
The most abstract class for all data expressions. A _DataExpr is a condition expression for a single data value. If the data value to be represent is a pure value (literal value, like string/number/Date/...) this _DataExpr is nothing more than a simple wrapper. If the data value to be represent is in a store, then _DataExpr is responsible to extract it from the store when this condition is applied to a data row. private fields: _value: anything _colArg: anything
