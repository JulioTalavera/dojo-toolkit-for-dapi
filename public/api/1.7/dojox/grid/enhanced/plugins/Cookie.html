Usage:

var foo = new dojox.grid.enhanced.plugins.Cookie(/* */ grid, /* */ args);

This plugin provides a way to persist some grid features in cookie. Default persistable features are: column width: "columnWidth" (handler name) column order: "columnOrder" sorting order: "sortOrder"

Grid users can define new persistable features by calling the following before grid is initialized (that is, during "preInit"); grid.addCookieHandler({ name: "a name for the new persistable feature", onLoad: function(savedObject, grid){ //load the cookie. }, onSave: function(grid){ //save the cookie. } });
