dojox.grid._LazyTreeGridCache (version 1.7)

Object » dojox.grid._LazyTreeGridCache
dojo.require("dojox.grid.LazyTreeGrid");
Defined in dojox/grid/LazyTreeGrid.js

Usage:

var foo = new dojox.grid._LazyTreeGridCache();

An internal object used to cache the tree path and open state of each item. The form of the cache items would be an object array: [{opened: true/false, treePath: [level0 parent id, level1 parent id, ...]}]

Examples:

Example 1

[{opened: true, treePath: []}, 
	{opened: false, treePath: ["root0"]},
	{opened: false, treePath: ["root0"]},
	{opened: false, treePath: []},
	...]
