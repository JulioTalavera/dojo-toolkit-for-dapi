Method Summary

  • _addItemClass(node, type) adds a class with prefix "dojoDndItem"
  • _changeState(type, newState) changes a named state to new state value
  • _conservativeBBLogic(node, left, top, right, bottom) returns Boolean logic which determines whether a node is bounded by the left,top,right,bottom parameters. This function returns true only if the coordinates of the node parameter are fully encompassed by the box determined by the left, top, right, bottom parameters.
  • _getChildByEvent(e) gets a child, which is under the mouse at the moment, or null
  • _getNodeId(nodeId, toNext) finds a next/previous node in relation to nodeId
  • _isBoundedByBox(node, left, top, right, bottom) figures out whether certain coodinates bound a particular dom node.
  • _liberalBBLogic(node, left, top, right, bottom) returns Boolean logic which determines whether a node is bounded by the left,top,right,bottom parameters. Allows for the case where any section of the box determined by the left,top,right,bottom parameters overlapping the coordinates of the node parameter constitutes a true return value
  • _normalizedCreator(item, hint) adds all necessary data to the output of the user-supplied creator function
  • _removeAnchor() returns self
  • _removeItemClass(node, type) removes a class with prefix "dojoDndItem"
  • _removeSelection() returns self unselects all items
  • clearItems() removes all data items from the map
  • constructor(node, params) returns dojox.dnd.Selector constructor of the Selector
  • delItem(key) removes a data item from the map by its key (id)
  • deleteSelectedNodes() returns self deletes all selected items
  • deselectNode(node) returns self deselects a node
  • destroy() prepares the object to be garbage-collected
  • forInItems(f, o) returns Object iterates over a data map skipping members that are present in the empty object (IE and/or 3rd-party libraries).
  • forInSelectedItems(f, o) iterates over selected items; see `dojo.dnd.Container.forInItems()` for details
  • getAllNodes() returns NodeList returns a list (an array) of all valid child nodes
  • getItem(key) returns dojo.dnd.Item returns a data item by its key (id)
  • getSelectedNodes() returns NodeList returns a list (an array) of selected nodes
  • insertNodes(addSelected, data, before, anchor) returns self inserts new data items (see `dojo.dnd.Container.insertNodes()` method for details)
  • isSelected(node) returns Boolean checks if node is selected
  • markupFactory(params, node, ctor)
  • selectAll() returns self selects all items
  • selectByBBox(left, top, right, bottom, add) returns self selects nodes by bounding box
  • selectNode(node, add) returns self selects a node
  • selectNone() returns self unselects all items
  • setItem(key, data) associates a data item with its key (id)
  • shift(toNext, add) shifts the currently selected dnd item forwards and backwards. One possible use would be to allow a user select different dnd items using the right and left keys.
  • startup() collects valid child items and populate the map
  • sync() returns self sync up the node list with the data map
