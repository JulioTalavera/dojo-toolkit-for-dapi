dojox.geo.openlayers.GreatCircle.toGeometryFeature
Defined by dojox.geo.openlayers.GreatCircle
GeometryFeature The geodetic line as a GeometryFeature

Create a geodetic line as a dojox.geo.openlayers.GeometryFeature between the point p1 ant the point p2. Result is a polyline approximation for which a new point is calculated every increment degrees.

ParameterTypeDescription
p1PointThe first point of the geodetic line. x and y fields are longitude and latitude in decimal degrees.
p2PointThe second point of the geodetic line. x and y fields are longitude and latitude in decimal degrees.
incrementFloatThe value at which a new point is computed.
