Usage:

var foo = new dojox.geo.openlayers.JsonImport(/* Object */ params);

This class loads JSON formated ShapeFile produced by the JSon Custom Map Converter. When loading the JSON file, it calls a iterator function each time a feature is read. This iterator function is provided as parameter to the constructor.

Returns dojox.geo.openlayers.JsonImport

Construct a new JSON importer with the specified parameters. These parameters are passed through an Object and include:

  • url : url
    • The url pointing to the JSON file to load.
  • nextFeature : function
    • The function called each time a feature is read. The function is called with a GeometryFeature as argument.
  • error : function
    • Error function called if something goes wrong.
