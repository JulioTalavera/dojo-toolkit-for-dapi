Defined by dojox.geo.charting
class to handle keyboard interactions on a dojox.geo.charting.Map widget The sections on the leading edge should receive the focus in response to a TAB event. Then use cursor keys to the peer sections. The cursor event should go the adjacent section in that direction. With the focus, the section zooms in upon SPACE. The map should zoom out on ESC. Finally, while it has the focus, the map should lose the focus on TAB.
