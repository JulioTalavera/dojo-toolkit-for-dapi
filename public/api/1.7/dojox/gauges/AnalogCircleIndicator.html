Usage:

var foo = new dojox.gauges.AnalogCircleIndicator();

An indicator for the AnalogGauge that draws a circle. The center of the circle is positioned on the circular gauge according to the value of the indicator. The circle has for radius the length of the indicator. This indicator is mainly used to draw round ticks for the scale.
