A widget you can use to embed either a Flash or Quicktime movie.

Examples:

Example 1

From markup: 
	<div dojoType="dojox.embed.Object" src="path/to/movie.swf"></div>

Example 2

Programmatic: 
	var mov=new dojox.embed.Object({
		src: "path/to/movie.swf"
	}, node);
