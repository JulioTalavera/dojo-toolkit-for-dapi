Usage:

var foo = new dojox.charting.action2d.TouchZoomAndPan(/* dojox.charting.Chart */ chart, /* */ plot, /* dojox.charting.action2d.__TouchZoomAndPanCtorArgs? */ kwArgs);

Create a touch zoom and pan action. You can zoom out or in the data window with pinch and spread gestures. You can scroll using drag gesture. Finally this is possible to navigate between a fit window and a zoom one using double tap gesture.

Property Summary

  • chart
  • defaultParams

Method Summary

  • _startScroll(axis)
  • connect() Connect this action to the chart. On Safari this adds a new glass view plot to the chart that's why Chart.render() must be called after connect.
  • constructor(chart, plot, kwArgs) returns dojox.charting.action2d.TouchZoomAndPan Create a new touch zoom and pan action and connect it.
  • destroy() Do any cleanup needed when destroying parent elements.
  • disconnect() Disconnect this action from the chart.

Event Summary

  • onDoubleTap(event) Called when double tap is performed on the chart.
  • onTouchEnd(event) Called when touch is ended on the chart.
  • onTouchMove(event) Called when touch is moved on the chart.
  • onTouchStart(event) Called when touch is started on the chart. we always want to be above regular plots and not clipped
