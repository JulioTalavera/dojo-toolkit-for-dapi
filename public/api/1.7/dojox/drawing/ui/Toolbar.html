Usage:

var foo = new dojox.drawing.ui.Toolbar(/* */ props, /* */ node);

Creates a GFX-based toobar that holds GFX-based buttons. Can be either created within the actual drawing or within a seperate DOM element. When within the drawing, the toolbar will cover a portion of the drawing; hence the option.

A Toolbar can be created programmtically or in markup. Currently markup is as a separate DOM element and programmtic is within the drawing. examples: dojo.connect(myDrawing, "onSurfaceReady", function(){ new dojox.drawing.ui.Toolbar({ drawing:myDrawing, tools:"all", plugs:"all", selected:"ellipse" }); });
