dojox.drawing.library.icons (version 1.7)

Object » dojox.drawing.library.icons
dojo.require("dojox.drawing.library.icons");
Defined in dojox/drawing/library/icons.js

Each icon name coresponds to a Stencil or a Plugin. One can be inserted into a button by adding the property: 'icon', which points to one of these Stencil objects.
