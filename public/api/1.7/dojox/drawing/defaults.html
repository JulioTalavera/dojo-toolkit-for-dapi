dojox.drawing.defaults (version 1.7)

Object » dojox.drawing.defaults
dojo.require("dojox.drawing.defaults");
Defined in dojox/drawing/defaults.js

This object contains defaults for objects used in Drawing. To change one item's style, use item.attr(); To change all these styles, create a copy of this file and point to it in the Drawing properties:

See: Drawing.changeDefaults

Determines whether in draw or edit mode (whether stencils are clickable. If clickMode is false, the original functionality of silently switching between select modes is enabled. If clickMode is true, it allows powerpoint- like functionality. Clickable is used by powerpoint to distinguish when things can be selected and when they can't

Property Summary

  • dojox.drawing.defaults.anchors Style for the anchor resize-points not really an anchor prop
  • dojox.drawing.defaults.arrows Size of arrows on vectors. length is in pixels width is actually an angle but is close to pixels in size
  • dojox.drawing.defaults.button
  • dojox.drawing.defaults.disabled Disabled or "locked" or "fade" style of all shapes
  • dojox.drawing.defaults.highlighted Highlighted style of all shapes NOT CURRENTLY BEING USED
  • dojox.drawing.defaults.hitHighlighted Highlighted style of a hit area
  • dojox.drawing.defaults.hitNorm Normal style of a hit area
  • dojox.drawing.defaults.hitSelected Selected style of a hit area
  • dojox.drawing.defaults.norm Normal style of all shapes will get overridden by above andes styles square, butt, round
  • dojox.drawing.defaults.selected Selected style of all shapes styles not shown will used from norm
  • dojox.drawing.defaults.text Style of text
  • dojox.drawing.defaults.textDisabled Style of disabled text
  • dojox.drawing.defaults.textMode These styles apply to the containing text box (edit mode), and not the text itself
