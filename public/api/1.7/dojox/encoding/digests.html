Method Summary

  • dojox.encoding.digests.MD5(data, outputType) returns word[]|string
  • dojox.encoding.digests.SHA1(data, outputType) returns word[]|string Computes the SHA1 digest of the data, and returns the result according to output type.
  • dojox.encoding.digests.addWords(a, b) returns word add a pair of words together with rollover
  • dojox.encoding.digests.stringToWord(s) returns word[] convert a string to a word array
  • dojox.encoding.digests.wordToBase64(wa) returns string convert an array of words to base64 encoding, should be more efficient than using dojox.encoding.base64
  • dojox.encoding.digests.wordToHex(wa) returns string convert an array of words to a hex tab
  • dojox.encoding.digests.wordToString(wa) returns string convert an array of words to a string
