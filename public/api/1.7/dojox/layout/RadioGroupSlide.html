dojox.layout.RadioGroupSlide (version 1.7)

Object » RadioGroup » dojox.layout.RadioGroupSlide
dojo.require("dojox.layout.RadioGroup");

Usage:

var foo = new dojox.layout.RadioGroupSlide();

An extension on a stock RadioGroup widget, sliding the pane into view from being hidden. The entry direction is randomized on each view

Method Summary

  • _hideChild(page) reset the position of the hidden pane out of sight
  • _positionChild(page) set the child out of view immediately after being hidden
  • _showChild(page) Slide in the selected child widget
  • constructor() returns dojox.layout.RadioGroupSlide A reference to the constructor function for this object.
  • easing() A hook to override the default easing of the pane slides.
