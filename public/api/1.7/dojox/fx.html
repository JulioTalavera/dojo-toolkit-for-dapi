dojox.fx (version 1.7)

Object » dojox.fx

A package of animations, and FX-related code, extending Dojo Core fx. Including this package includes all the Base and Core fx packages.

Property Summary

  • dojox.fx.ext-dojo Direct extensions to dojo.fx namespace

Method Summary

  • dojox.fx.Shadow() Adds a drop-shadow to a node.
  • dojox.fx._Complex(options) A class that takes a complex property such as clip style: rect(10px 30px 10px 50px), and breaks it into seperate animatable units. The object has a getValue() that will return a string with the modified units.

	});
