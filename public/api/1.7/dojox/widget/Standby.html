dojox.widget.Standby (version 1.7)

Object » _Widget » dojox.widget.Standby
dojo.require("dojox.widget.Standby");
Defined in dojox/widget/Standby.js

Usage:

var foo = new dojox.widget.Standby();

A widget designed to act as a Standby/Busy/Disable/Blocking widget to indicate a particular DOM node is processing and cannot be clicked on at this time. This widget uses absolute positioning to apply the overlay and image.

Property Summary

  • _textNode The div to attach text/HTML in the overlay center item.
  • target The target to overlay when active. Can be a widget id, a dom id, or a direct node reference.

Method Summary

  • _cloneStyles(list) Internal function to clone a set of styles from the target to the underlay.
  • _disableOverflow() Function to disable scrollbars on the body. Only used if the overlay targets the body or the document.
  • _enableOverflow() Function to restore scrollbars on the body. Only used if the overlay targets the body or the document.
  • _fadeIn() Internal function that does the opacity style fade in animation.
  • _fadeOut() Internal function that does the opacity style fade out animation.
  • _ignore(e) Function to ignore events that occur on the overlay. event: Event The event to halt
  • _scrollerWidths() This function will calculate the size of the vertical and horizontaol scrollbars.
  • _setCenterIndicatorAttr(indicator) Function to allow widget.attr to set the node used for the center indicator, either the image or the text.
  • _setColorAttr(c) Function to allow widget.attr to set the color used for the translucent div overlay.
  • _setImageAttr(url) Function to allow widget.attr to set the url source for the center image text: String The url to set for the image.
  • _setImageTextAttr(text) Function to allow widget.attr to set the ALT text text displayed for the image (if using image center display).
  • _setTextAttr(text) Function to allow widget.attr to set the text displayed in center if using text display.
  • _size() Internal function that handles resizing the overlay and centering of the image on window resizing.
  • constructor() returns dojox.widget.Standby A reference to the constructor function for this object.
  • hide() Function to hide the blocking overlay and status icon or text.
  • isVisible() returns boolean Helper function so you can test if the widget is already visible or not.
  • show() Function to display the blocking overlay and busy/status icon or text.
  • startup(args) Over-ride of the basic widget startup function. Configures the target node and sets the image to use.
  • uninitialize() Over-ride to hide the widget, which clears intervals, before cleanup.

Event Summary

  • onHide() Event that fires when the display of the Standby completes.
  • onShow() Event that fires when the display of the Standby completes.
