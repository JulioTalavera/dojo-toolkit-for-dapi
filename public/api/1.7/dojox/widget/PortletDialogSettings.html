dojox.widget.PortletDialogSettings (version 1.7)

Object » dijit._Container » dojox.widget.PortletSettings » dojox.widget.PortletDialogSettings
dojo.require("dojox.widget.Portlet");
Defined in dojox/widget/Portlet.js

Usage:

var foo = new dojox.widget.PortletDialogSettings(/* */ props, /* */ node);

A settings widget to be used with a dojox.widget.Portlet, which displays the contents of this widget in a dijit.Dialog box.

Event Summary

  • _onError(type, err, consoleText)

  • _onShow() returns If child has an href, promise that fires when the load is complete Called when the ContentPane is made visible
  • _onUnloadHandler() This is called whenever the content is being unloaded
  • onContentError(error) Called on DOM faults, require faults etc. in content. In order to display an error message in the pane, return the error message from this method, as an HTML string. By default (if this method is not overriden), it returns nothing, so the error message is just printed to the console.
  • onDownloadEnd() Called when download is finished.
  • onDownloadError(error) Called when download error occurs. In order to display an error message in the pane, return the error message from this method, as an HTML string. Default behavior (if this method is not overriden) is to display the error message inside the pane.

  • onLoad(data) Event hook, is called after everything is loaded and widgetified
  • onUnload() Event hook, is called before old content is cleared
