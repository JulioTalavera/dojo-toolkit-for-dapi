dojox.mobile.ViewController (version 1.7)

Object » dojox.mobile.ViewController

Usage:

var foo = new dojox.mobile.ViewController();

This class listens to the "startTransition" events and performs view transitions. If the transition destination is an external view specified with the url parameter, retrieves the view content and parses it to create a new target view.

Method Summary

  • _instantiate(obj, node, parent) Given the evaluated json data, does the same thing as what the parser does.
  • _parse(text, id) Parses the given view content.
