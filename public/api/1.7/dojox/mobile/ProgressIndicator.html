dojox.mobile.ProgressIndicator (version 1.7)

Usage:

var foo = new dojox.mobile.ProgressIndicator();

ProgressIndicator is a round spinning graphical representation that indicates the current task is on-going.

Property Summary

  • colors An array of indicator colors.

Method Summary

  • constructor() returns dojox.mobile.ProgressIndicator A reference to the constructor function for this object.
  • setImage(file) Sets an indicator icon image file (typically animated GIF). If null is specified, restores the default spinner.
  • start() Starts the ProgressIndicator spinning.
  • stop() Stops the ProgressIndicator spinning.
