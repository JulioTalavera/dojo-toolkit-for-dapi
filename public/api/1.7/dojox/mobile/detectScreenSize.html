dojox.mobile.detectScreenSize (version 1.7)

Object » dojox.mobile.detectScreenSize
dojo.require("dojox.mobile.common");
Defined in dojox/mobile/common.js

Usage:

var foo = new dojox.mobile.detectScreenSize(/* Boolean? */ force);

Detects the screen size and determines if the screen is like phone or like tablet. If the result is changed, it sets either of the following css class to <html> - 'djphone' - 'djtablet' and it publishes either of the following events. - '/dojox/mobile/screenSize/phone' - '/dojox/mobile/screenSize/tablet'
