dojox.gfx (version 1.7)

Object » dojox.gfx

dojox.gfx is an advanced API providing normalized vector drawing in a variety of browsers. It has individual renderers for SVG, VML, Canvas, and Silverlight.

Property Summary

  • dojox.gfx._base
  • dojox.gfx.arc This module contains the core graphics Arc functions.
  • dojox.gfx.canvas This the graphics rendering bridge for W3C Canvas compliant browsers. Since Canvas is an immediate mode graphics api, with no object graph or eventing capabilities, use of this module alone will only add in drawing support. The additional module, canvasWithEvents extends this module with additional support for handling events on Canvas. By default, the support for events is now included however, if only drawing capabilities are needed, canvas event module can be disabled using the dojoConfig option, canvasEvents:true|false. The id of the Canvas renderer is 'canvas'. This id can be used when switch Dojo's graphics context between renderer implementations. See dojox.gfx._base switchRenderer API.
