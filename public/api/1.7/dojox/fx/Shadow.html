dojox.fx.Shadow (version 1.7)

Object » Widget » dojox.fx.Shadow
dojo.require("dojox.fx.Shadow");
Defined in dojox/fx/Shadow.js

Usage:

var foo = new dojox.fx.Shadow();

Adds a drop-shadow to a node.

Examples:

Example 1

	// add drop shadows to all nodes with class="hasShadow"
	dojo.query(".hasShadow").forEach(function(n){
		var foo = new dojox.fx.Shadow({ node: n });
		foo.startup();
	});

Method Summary

  • _makePiece(name, vertAttach, vertCoord, horzAttach, horzCoord, sizing) append a shadow pieces to the node, and position it
  • constructor() returns dojox.fx.Shadow A reference to the constructor function for this object.
  • resize(args) Resizes the shadow based on width and height.
  • setDisabled(disabled) enable / disable the shadow
  • setOpacity(n, animArgs) set the opacity of the underlay note: does not work in IE? FIXME.
  • startup() Initializes the shadow.
