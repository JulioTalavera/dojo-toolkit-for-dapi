dijit._TemplatedMixin (version 1.7)

Object » dijit._TemplatedMixin

Usage:

var foo = new dijit._TemplatedMixin();

Mixin for widgets that are instantiated from a template

Method Summary

  • _attachTemplateNodes(rootNode, getAttrFunc) Iterate through the template and attach functions and nodes accordingly. Alternately, if rootNode is an array of widgets, then will process data-dojo-attach-point etc. for those widgets.
  • _beforeFillContent()
  • _fillContent(source) Relocate source contents to templated container node. this.containerNode must be able to receive children, or exceptions will be thrown.
  • _stringRepl(tmpl) Does substitution of ${foo} type properties in template string
  • buildRendering() Construct the UI for this widget from a template, setting this.domNode.
  • constructor() returns dijit._TemplatedMixin A reference to the constructor function for this object.
  • destroyRendering()

Methods

_attachTemplateNodes
Defined by dijit._TemplatedMixin

Map widget properties and functions to the handlers specified in the dom node and it's descendants. This function iterates over all nodes and looks for these properties:

  • dojoAttachPoint/data-dojo-attach-point
  • dojoAttachEvent/data-dojo-attach-event
ParameterTypeDescription
rootNodeDomNode|Widget[]the node to search for properties. All children will be searched.
getAttrFuncFunctiona function which will be used to obtain property for a given DomNode/Widget
