dijit.form._FormWidgetMixin (version 1.7)

Object » dijit.form._FormWidgetMixin

Usage:

var foo = new dijit.form._FormWidgetMixin();

Represents a single HTML element. All these widgets should have these attributes just like native HTML input elements. You can set them during widget construction or afterwards, via dijit._Widget.attr.

They also share some common methods.
