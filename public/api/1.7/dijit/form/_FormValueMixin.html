dijit.form._FormValueMixin (version 1.7)

Object » dijit.form._FormWidgetMixin » dijit.form._FormValueMixin

Usage:

var foo = new dijit.form._FormValueMixin();

Each _FormValueMixin represents a single input value, and has a (possibly hidden) element, to which it serializes it's input value, so that form submission (either normal submission or via FormBind?) works as expected.
