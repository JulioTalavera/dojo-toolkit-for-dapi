A mixin for all number textboxes

Property Summary

  • dijit.form.NumberTextBoxMixin.textbox
  • editOptions Properties to mix into constraints when the value is being edited. This is here because we edit the number in the format "12345", which is different than the display value (ex: "12,345")

Method Summary

  • _formatter(value, options) returns String _formatter() is called by format(). It's the base routine for formatting a number, as a string, for example converting 12345 into "12,345".
  • _getDisplayedValueAttr()

  • _parser(value, constraints) returns Number Parses the string value as a Number, according to constraints.
  • _setBlurValue()
  • _setConstraintsAttr(constraints)
  • _setValueAttr(value, priorityChange, formattedValue) Hook so set('value', ...) works.
  • constructor() returns dijit.form.NumberTextBoxMixin A reference to the constructor function for this object.
  • filter(value) returns set('value', null||''|undefined) should fire onChange(NaN) This is called with both the display value (string), and the actual value (a number). When called with the actual value it does corrections so that '' etc. are represented as NaN. Otherwise it dispatches to the superclass's filter() method. See `dijit.form.TextBox.filter` for more details.
  • format(value, constraints) Formats the value as a Number, according to constraints.
