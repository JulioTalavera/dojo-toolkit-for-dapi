Usage:

var foo = new dijit.tree._dndContainer(/* Node */ tree, /* dijit.tree.__SourceArgs */ params);

This is a base class for dijit.tree._dndSelector, and isn't meant to be used directly. It's modeled after dojo.dnd.Container.
