connect
Defined by dijit._OnDijitClickMixin
A handle that can be passed to `disconnect` in order to disconnect before the widget is destroyed.

Provide widget-specific analog to connect.connect, except with the implicit use of this widget as the target object. This version of connect also provides a special "ondijitclick" event which triggers on a click or space or enter keyup. Events connected with this.connect are disconnected upon destruction.
