dojoConfig (version 1.7)

Object » dojoConfig
dojo.require("dojo._base.config");
Defined in dojo/_base/config.js

Application code can set the global 'dojoConfig' prior to loading the library to control certain global settings for how dojo works.

Method Summary

  • dojoConfig.addOnLoad() or Array Adds a callback via dojo.addOnLoad. Useful when Dojo is added after the page loads and djConfig.afterOnLoad is true. Supports the same arguments as dojo.addOnLoad. When using a function reference, use `djConfig.addOnLoad = function(){};`. For object with function name use `djConfig.addOnLoad = [myObject, "functionName"];` and for object with function reference use `djConfig.addOnLoad = [myObject, function(){}];`
