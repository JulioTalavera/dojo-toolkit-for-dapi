Method Summary

  • dojo.dnd.AutoSource(node, params) a source that syncs its DnD nodes by default
  • dojo.dnd.Avatar(manager) Object that represents transferred DnD items visually manager: Object a DnD manager object
  • dojo.dnd.Container(node, params) a Container object, which knows when mouse hovers over it, and over which element it hovers
  • dojo.dnd.Item() Represents (one of) the source node(s) being dragged. Contains (at least) the "type" and "data" attributes.
  • dojo.dnd.Manager() the manager of DnD operations (usually a singleton)
  • dojo.dnd.Moveable(node, params) an object, which makes a node moveable
  • dojo.dnd.Mover(node, e, host) an object which makes a node follow the mouse, or touch-drag on touch devices. Used as a default mover, and as a base class for custom movers.
  • dojo.dnd.Selector(node, params) a Selector object, which knows how to select its children
  • dojo.dnd.Source(node, params) a Source object, which can be used as a DnD source, or a DnD target
  • dojo.dnd.Target(node, params) a Target object, which can be used as a DnD target
  • dojo.dnd.TimedMoveable(node, params) A specialized version of Moveable to support an FPS throttling. This class puts an upper restriction on FPS, which may reduce the CPU load. The additional parameter "timeout" regulates the delay before actually moving the moveable object.
  • dojo.dnd.__ContainerArgs()
  • dojo.dnd.__MoveableArgs()
  • dojo.dnd.__SelectorArgs()
  • dojo.dnd.__SourceArgs() a dict of parameters for DnD Source configuration. Note that any property on Source elements may be configured, but this is the short-list
  • dojo.dnd.__TimedMoveableArgs()
  • dojo.dnd._createNode(tag) returns Function|Node returns a function, which creates an element of given tag (SPAN by default) and sets its innerHTML to given text
  • dojo.dnd._createSpan(text) returns Node creates a SPAN element with given text as its innerHTML
  • dojo.dnd._createTrTd(text) returns Node creates a TR/TD structure with given text as an innerHTML of TD
  • dojo.dnd._defaultCreator(node) returns Function takes a parent node, and returns an appropriate creator function
  • dojo.dnd.autoScroll(e) a handler for onmousemove event, which scrolls the window, if necesary
  • dojo.dnd.autoScrollNodes(e) a handler for onmousemove event, which scrolls the first avaialble Dom element, it falls back to dojo.dnd.autoScroll()
  • dojo.dnd.getUniqueId() returns a unique string for use with any DOM element
  • dojo.dnd.isFormElement(e) returns Boolean returns true if user clicked on a form element
  • dojo.dnd.manager() returns Object Returns the current DnD manager. Creates one if it is not created yet.
