dojo.touch (version 1.7)

Object » dojo.touch
dojo.require("dojo.touch");
Defined in dojo/touch.js

This module provides unified touch event handlers by exporting press, move, release and cancel which can also run well on desktop. Based on http://dvcs.w3.org/hg/webevents/raw-file/tip/touchevents.html

Examples:

Example 1

1. Used with dojo.connect() 
	dojo.connect(node, dojo.touch.press, function(e){});
	dojo.connect(node, dojo.touch.move, function(e){});
	dojo.connect(node, dojo.touch.release, function(e){});
	dojo.connect(node, dojo.touch.cancel, function(e){});
2. Used with dojo.on 
	define(["dojo/on", "dojo/touch"], function(on, touch){
		on(node, touch.press, function(e){});
		on(node, touch.move, function(e){});
		on(node, touch.release, function(e){});
		on(node, touch.cancel, function(e){});
3. Used with dojo.touch.* directly 
	dojo.touch.press(node, function(e){});
	dojo.touch.move(node, function(e){});
	dojo.touch.release(node, function(e){});
	dojo.touch.cancel(node, function(e){});

Method Summary

  • dojo.touch.cancel(node, listener) Register a listener to 'touchcancel'|'mouseleave' for the given node
  • dojo.touch.move(node, listener) Register a listener to 'touchmove'|'mousemove' for the given node
  • dojo.touch.press(node, listener) Register a listener to 'touchstart'|'mousedown' for the given node
  • dojo.touch.release(node, listener) Register a listener to 'touchend'|'mouseup' for the given node
